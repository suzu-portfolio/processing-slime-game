Processing Slime Game

概要

Processing（Java）を用いて制作したアクションゲームです。
プレイヤーの色と味方のスライムをキャッチすることで大きく成長し、スコアを獲得します。
敵スライムを取るとスコアは0になり、成長したスライムは元の大きさに戻る仕組みになっています。

ゲーム内容

・左右移動でスライムをキャッチ
・味方のスライムを吸収することでプレイヤースライムが成長、敵スライムを吸収すると初期サイズに戻る
・味方のスライムで加点、敵スライムでスコア0

工夫した点

・スピードアップ要素や、敵スライムによるスコア・スライムサイズの変化によって状況判断が必要なゲーム性にした
・スコアの加減点ロジックを設計し、単純な操作でも戦略性が生まれるよう工夫した
・クラス（Player / EnemySlime など）ごとに役割を分け、コードの可読性と拡張性を意識した設計にした
・アニメーション（ジャンプ動作など）を加え、視覚的な動きを強化した
・最終スコアによるエンド分岐を実装し、何度も遊べびたいと思えるようなデザインを行った

使用技術

・Processing（Java）

実行方法

Processingを起動
本リポジトリのファイルをダウンロード
main.pde を開いて実行
